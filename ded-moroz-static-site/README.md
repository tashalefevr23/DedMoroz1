# Дед Мороз и Снегурочка — статический сайт
Готов к развёртыванию на Vercel.

## Деплой
1. Зайдите на https://vercel.com → New Project → Import → Upload Folder.
2. Выберите папку проекта (эту папку).
3. Нажмите Deploy. Готово!

## Структура
- index.html — весь сайт (таймер считает до ближайшего Нового года)
- /images/hero.jpg — фон хиро-секции
